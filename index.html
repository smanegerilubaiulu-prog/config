How to: Konfigurasi Ujian

Untuk alasan integritas, ujian hanya dapat dilaksanakan menggunakan aplikasi di bawah ini. Silahkan pilih perangkat yang kamu gunakan untuk ujian:

Download Extraordinary CBT

Cara Menggunakan (Android)

1. Download dan install aplikasi.
2. Buka aplikasi tersebut dan izinkan mode jangan ganggu.
3. Scan QR Code di atas untuk masuk ke ujian.

Penting:
• Jangan keluar dari aplikasi saat ujian.
• Pastikan baterai dan internet stabil.
• Tidak ada pembukaan blokir.
Download SafeExamBrowser Download File Ujian (.seb)

Cara Menggunakan (Apple)

1. Download dan install SafeExamBrowser.
2. Download file ujian (.seb).
3. Buka file ujian (.seb) tersebut untuk masuk ke ujian atau bagikan ke SEB.

Penting:
• Jangan menutup aplikasi selama ujian.
• Jangan membuka aplikasi lain.
• Tidak ada pembukaan blokir.
